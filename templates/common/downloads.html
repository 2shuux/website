Downloads

Lutris should be compatible with most up-to-date Linux distributions, it relies on the Python language and Gnome 3 libraries (but will run fine under any desktop environmment). The following distributions are currently supported: Ubuntu 14.04, Debian 8 (Jessie), Fedora 20, OpenSUSE 13.1, ArchLinux and Gentoo.

    • Ubuntu
    • Debian

    Download lutris_{{ version }}_all.deb and install it with the Software Center or gdebi.

    • Fedora
    • OpenSuse

    Download lutris-{{ version }}-3.fc20.noarch.rpm and install it with rpm.

    • ArchLinux

    You can get Lutris from the AUR repositories: https://aur.archlinux.org/packages/lutris/.
    or get the version from git: https://aur.archlinux.org/packages/lutris-git/. (may be unstable, use with caution)

    • Gentoo

    Lutris is available in the official Gentoo repositories: http://packages.gentoo.org/package/games-util/lutris.

    • tar.gz

    Lutris is available as a tar.gz package. Install it by extracting the archive and running as root: python setup.py install
    You'll need to make sure you have the necessary dependencies: PyGObject3 and PyYAML.
    On Slackware systems, you can those at: 

            http://slackbuilds.org/repository/14.1/python/pygobject3/
        http://slackbuilds.org/repository/14.1/libraries/PyYAML/

    • Github

    Lutris is constantly evolving. If you want to try out the latest features or help with the development, you can get Lutris from Github: git clone http://github.com/lutris/lutris.git
    Please report all problems on the bugtracker or on IRC (#lutris on Freenode)
