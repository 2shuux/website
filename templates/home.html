What is Lutris?

Lutris is a gaming platform for Linux.
Thanks to community-provided installers, you spend less time setting games up, and more time playing.
Free and Open-Source games are available right from this website.

Lutris is open!

Lutris offers you a completely Open-Source client and enforces absolutely no vendor lock-in.
The client stays completely usable when offline or without access to the website.

Thousands of games!

From the first games of your childhood to the latest indie game, games for Linux, Windows, Amiga, Super Nintendo and many more are supported, if it's able to run on Linux, it can be run with Lutris. This is a project meant to help video game preservation!
